Admin: Books Management

Endpoint untuk mengelola data buku. Semua endpoint di sini membutuhkan otentikasi dengan role **Admin**.

Get All Books (Admin)

GET /api/v2/admin/books Admin Only

Mengambil daftar semua buku yang ada di sistem.

Create Book

POST /api/v2/admin/books Admin Only

Menambahkan buku baru. Endpoint ini dapat menerima **JSON** (tanpa gambar) atau **Form-Data** (jika ingin mengunggah gambar sampul).

Form-Data Fields:

  • title (text): Judul buku.
  • author (text): Penulis buku.
  • price (text): Harga dalam angka.
  • stock (text): Jumlah stok.
  • published_year (text): Tahun terbit.
  • category_id (text): UUID dari kategori.
  • description (text, optional): Deskripsi buku.
  • cover_image (file, optional): File gambar sampul.

Get Book Detail (Admin)

GET /api/v2/admin/books/{id} Admin Only

Mengambil detail satu buku berdasarkan UUID-nya.

URL Parameters:

  • id (uuid): UUID dari buku yang ingin dilihat.

Update Book

PUT /api/v2/admin/books/{id} Admin Only

Memperbarui data buku. Ganti `{id}` dengan UUID buku. Endpoint ini juga dapat menerima **JSON** atau **Form-Data**.

Delete Book

DELETE /api/v2/admin/books/{id} Admin Only

Menghapus buku berdasarkan UUID-nya.
