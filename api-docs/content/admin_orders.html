Admin: Orders Management

Endpoint untuk mengelola pesanan dari semua pengguna. Semua endpoint di sini membutuhkan otentikasi dengan role **Admin**.

Get All Orders

GET /api/v2/admin/orders Admin Only

Mengambil daftar semua pesanan. Dapat difilter berdasarkan status.

Query Parameters:

  • status (string, optional): Filter pesanan berdasarkan status. Nilai yang mungkin: `pending`, `diproses`, `dikirim`, `selesai`, `batal`. Contoh: ?status=diproses

Get Order Detail (Admin)

GET /api/v2/admin/orders/{id} Admin Only

Mengambil detail lengkap satu pesanan berdasarkan UUID-nya.

Update Order Status

PUT /api/v2/admin/orders/{id}/status Admin Only

Memperbarui status pesanan. Ganti `{id}` dengan UUID pesanan. Request body harus berupa **JSON**.

Request Body:

{
    "status": "dikirim"
}
