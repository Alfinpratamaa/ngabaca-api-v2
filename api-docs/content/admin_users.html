Admin: Users Management

Endpoint untuk mengelola data pengguna. Semua endpoint di sini membutuhkan otentikasi dengan role **Admin**.

Get All Users

GET /api/v2/admin/users Admin Only

Mengambil daftar semua pengguna yang terdaftar di sistem.

Update User Role

PUT /api/v2/admin/users/{id} Admin Only

Memperbarui peran (role) seorang pengguna. Ganti `{id}` dengan UUID pengguna. Request body harus berupa **JSON**.

Request Body:

{
    "role": "admin"
}
