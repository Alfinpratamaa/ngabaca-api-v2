Authentication

Register User

POST /api/v2/auth/register

Mendaftarkan akun pengguna baru. Request body harus berupa **JSON**.

Request Body:

{
    "name": "John Doe",
    "email": "john.doe@example.com",
    "password": "password123"
}

Login User

POST /api/v2/auth/login

Mengautentikasi pengguna dan mengembalikan JWT token. Request body harus berupa **JSON**.

Request Body:

{
    "email": "john.doe@example.com",
    "password": "password123"
}
