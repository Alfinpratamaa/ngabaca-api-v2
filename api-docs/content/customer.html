Customer Actions

Endpoint untuk aksi yang dilakukan oleh pelanggan, seperti melihat pesanan, checkout, dan memberi ulasan. Semua endpoint di sini membutuhkan otentikasi (Bearer Token).

Get My Orders

GET /api/v2/customer/orders Protected

Mengambil riwayat pesanan dari pengguna yang sedang login.

Get Order Detail

GET /api/v2/customer/orders/{id} Protected

Mengambil detail satu pesanan spesifik milik pengguna.

URL Parameters:

  • id (uuid): UUID dari pesanan yang ingin dilihat.

Checkout

POST /api/v2/customer/checkout Protected

Membuat pesanan baru dan memulai sesi pembayaran Midtrans. Request body harus berupa **JSON**.

Request Body:

{
    "items": [
        {
            "book_id": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
            "quantity": 2
        }
    ],
    "shipping_address": "Jl. Merdeka No. 10, Bandung",
    "notes": "Tolong bungkus dengan rapi."
}

Create a Review

POST /api/v2/customer/books/{id}/reviews Protected

Menambahkan ulasan dan rating pada sebuah buku. Request body harus berupa **JSON**.

URL Parameters:

  • id (uuid): UUID dari buku yang akan diulas.

Request Body:

{
    "rating": 5,
    "comment": "Bukunya sangat bagus dan direkomendasikan!"
}
