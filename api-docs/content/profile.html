User Profile

Endpoint untuk mengelola profil pengguna yang sedang login. Semua endpoint di sini membutuhkan otentikasi (Bearer Token).

Get My Profile

GET /api/v2/me Protected

Mengambil detail profil dari pengguna yang sedang login.

Update My Profile

PUT /api/v2/me Protected

Memperbarui nama atau nomor telepon pengguna. Request body harus berupa **JSON**.

Request Body:

{
    "name": "Nama Baru Saya",
    "phone_number": "081234567891"
}

Upload Avatar

POST /api/v2/me/avatar Protected

Mengunggah gambar profil baru. Request body harus berupa **Form-Data**.

Form-Data Fields:

  • key: avatar, type: File, value: (pilih file gambar Anda dari komputer)
