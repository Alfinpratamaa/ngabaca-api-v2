Public Endpoints

Endpoint berikut dapat diakses oleh siapa saja tanpa perlu otentikasi.

Get Book Catalog

GET /api/v2/catalog

Mengambil daftar semua buku yang tersedia.

Get Book Detail

GET /api/v2/book/{slug}

Mengambil detail satu buku berdasarkan slug-nya.

URL Parameters:

  • slug (string): Slug unik dari buku yang ingin dilihat.

Get All Categories

GET /api/v2/categories

Mengambil daftar semua kategori buku.

Search Books

GET /api/v2/search

Mencari buku berdasarkan judul atau penulis.

Query Parameters:

  • q (string): Kata kunci yang ingin dicari. Contoh: ?q=pemrograman

Get Book Reviews

GET /api/v2/books/{id}/reviews

Mengambil semua ulasan untuk buku tertentu.

URL Parameters:

  • id (uuid): UUID dari buku yang ulasannya ingin dilihat.
