Webhook

Endpoint yang tidak dipanggil oleh pengguna, melainkan oleh layanan eksternal.

Midtrans Notification

POST /midtrans/notification

Endpoint ini menerima notifikasi status pembayaran dari Midtrans. Midtrans akan mengirim request ke URL ini setiap kali ada perubahan status pada transaksi. Backend akan memvalidasi dan memperbarui status pesanan secara otomatis.

Lihat dokumentasi resmi Midtrans untuk format payload yang lengkap.
